# Directory Listing

## Tech Stack:

### Frontend:
- React.Js
- Redux
- React-redux
- Redux toolkit
- Chakra UI
- React-router-dom

## Setup Instructions
1. Clone the repository from GitHub.
2. Navigate to the project directory in the terminal.
3. Install dependencies:
```bash
npm install
```
4. Start the frontend development server:
```bash
npm run start
```
5. Access the application in your web browser.
